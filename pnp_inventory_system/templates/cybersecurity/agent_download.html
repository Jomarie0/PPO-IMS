Installation Instructions
Step 1: Generate Token

Each branch requires a unique authentication token. Contact your system administrator to generate a token for your branch.

Step 2: Install Agent

Download and run the installer with administrator privileges. Follow the on-screen instructions to complete installation.

Step 3: Configure Agent

Enter your branch token when prompted during setup. The agent will automatically sync with the central server.

Step 4: Verify Installation

Check the agent status in system tray. Green indicator means successful connection to server.

Branch Token Management
Administrators: Use the Django admin interface to generate and manage branch tokens. Navigate to Cybersecurity → Branch Agent Tokens in the admin panel.
