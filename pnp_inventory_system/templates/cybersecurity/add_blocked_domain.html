Domain Blocking Information
What happens when a domain is blocked?
  • The domain is added to the blocklist distributed to all agents
  • Agents prevent access to blocked domains at network level
  • Block is logged and can be audited
  • Block can be toggled on/off at any time
Agent Integration

Blocked domains are enforced through the security agents:

  • Agents sync blocklist every 5 minutes
  • Domain blocking works at DNS/hosts file level
  • Blocks apply to all browsers and applications
  • Bypass requires administrator privileges
Important: Domain blocking affects all workstations with the agent installed.
Tip: Use subdomain blocking carefully as it may affect legitimate services.
