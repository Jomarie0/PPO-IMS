IP Blocking Information
What happens when an IP is blocked?
  • All requests from the blocked IP receive a 403 Forbidden response
  • The IP cannot access any part of the system
  • Block is logged and can be audited
  • Block can be toggled on/off at any time
Automatic IP Blocking

The system automatically blocks IPs that attempt too many failed logins:

  • 5 failed attempts within 1 hour
  • Automatic block with reason "Too many failed login attempts"
  • System-generated blocks can be managed manually
Important: Be careful when blocking IP addresses to avoid blocking legitimate users.
